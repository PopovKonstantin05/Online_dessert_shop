Онлайн-магазин десертов, реализованный с помощью React, JS, Redux Toolkit и моковыми данными, использующимися в качестве Backend.

Для запуска проекта сначала нужно запустить моковые данные, находящиеся в файле data.json командой json-server --watch data.json --port 3001.
После этого можно запускать сам проект - npm run start
